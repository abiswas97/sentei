# Design: light-contrast

Two palette values. ANSI 245 (#8a8a8a) on white is ~3.5:1 — below the 4.5:1
text floor; 243 (#767676) clears it. ANSI 166 (#d75f00) on white is ~3.2:1;
130 (#af5f00) reaches ~4.6:1 while staying recognizably orange. Verified
with a light-theme VHS tape (the v2 background report drives palette
selection automatically).
